Learning Git

Step 1: Right click a folder you would like your local repository to exist in.
Step 2: Click Git Bash (This opens a command window MINGW32 or something along those lines)
Step 3: Type git init which will initialize your Git repository in the selected folder you have chosen
Step 4: Type git add 'filename.ending' which will make your file ready to commit (This is called staging)
Step 5: Type git commit (This will pull up another command window.  Type your commit message which describes what changes you have made and then exit the window.  You will either have to 	confirm your commit or your commit will automatically be confirmed.
Step 6: To update your online repository on Github.com type git push
